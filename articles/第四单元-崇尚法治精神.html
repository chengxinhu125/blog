第七课：尊重自由平等

┌─────────────────────────────────────────┐
│           自由平等的价值观                │
├─────────────────────────────────────────┤
│  自由：珍视自由，敢于行使权利            │
│  平等：践行平等，反对歧视                │
│  法治：守护正义，促进公平                │
└─────────────────────────────────────────┘

📌 自由

• 自由不是想干什么就干什么
• 自由是宪法法律范围内的自由
• 珍视自由 = 依法行使权利 + 尊重他人自由

📌 平等

• 公民在法律面前一律平等
• 反对任何形式的歧视
• 平等 ≠ 平均，平等 ≠ 同等
  平等是同等情况同等对待，不同情况差别对待

📌 第八课：维护公平正义

公平：
  → 权利公平（机会同等）
  → 规则公平（制度保障）
  → 结果公平（利益均衡）

正义：
  → 正义是社会制度的重要价值
  → 正义是社会和谐的基本保障
  → 守护正义需要法治保障

💡 核心考点

公平包括：权利公平、规则公平、分配公平
正义的核心：程序正义 + 结果正义
如何守护正义：立法正义（科学立法）+ 执法正义（严格执法）
